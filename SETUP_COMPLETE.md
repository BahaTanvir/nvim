## 🚀 Quick Start Guide

### 1. **First Launch**
```bash
nvim
```
- Plugins will auto-install on first launch
- Wait for all installations to complete

### 2. **Health Check**
```vim
:checkhealth
```
- Verify all dependencies are working
- Fix any reported issues

### 3. **Essential First Commands**
```vim
:Telescope find_files    " Find files (or <leader>ff)
:Telescope live_grep     " Search in files (or <leader>fs)
:NvimTreeToggle         " File explorer (or <leader>ee)
:Lazy                   " Plugin manager
:Mason                  " LSP/tool installer
```

### 4. **AI Features**
```vim
" In insert mode:
<C-l>                   " Accept Copilot suggestion
<leader>ccq             " Quick AI chat
<leader>cce             " Explain code
```

### 5. **Session Management**
```vim
<leader>qs              " Restore last session
<leader>ql              " Load session
<leader>qd              " Don't save current session
```
